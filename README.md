## Descripción del Proyecto
El objetivo de esta tarea es realizar un **Análisis Exploratorio de Datos (EDA)** utilizando indicadores económicos, demográficos y sociales globales del año 2007. Se integran fuentes de datos adicionales sobre desigualdad (Índice de Gini) y emisiones fósiles de CO2 por nación.

## Análisis Realizados
1. **Estadísticas Descriptivas:** Limpieza de datos, manejo de valores nulos (imputación por media/mediana) y detección de outliers.
2. **Visualización de Datos:** Histogramas, boxplots y scatterplots para entender la distribución del PIB y otras variables.
3. **Correlaciones Económicas:** Comparación entre PIB per cápita (bienestar) y PIB total (escala).
4. **Análisis de Gini:** Cruce de datos para observar la relación entre desigualdad e indicadores de desarrollo.
5. **Impacto Ambiental:** Estudio de las emisiones de CO2 y cómo shocks económicos (Crisis Subprime 2008 y COVID-19 2020) afectaron la contaminación global.
